# intro
esto es un script automatico para tollear las votaciones de la pagina [https://yoparticipo.voto/] :)

# requerimientos
- python 3.8
- pipenv
- geckodriver
- firefox
# uso
una ves instalado los requerimientos

correr:

```
pipenv shell
pipenv install
python voto.py
```
